== Description ==

Attaches Marketo tracking code to website. Lets you map any form field to a specific field in marketo, 
including custom fields. NOTE: This is not a forms plugin. 
You use it in conjunction with your existing forms solution, or you can also use it with custom forms.

Now it will help you identify form fields on a page!

== Installation ==

1. Go to Plugins page in Wordpress
2. Select "Add New"
3. Select "Upload"
4. Browse for and select "wordpress-marketo-lead-tracking.zip"
5. Click "Install"
6. Activate plugin
